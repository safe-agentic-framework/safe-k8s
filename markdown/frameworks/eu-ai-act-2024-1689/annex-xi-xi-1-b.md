# EU AI Act 2024/1689 - Annex-XI:XI(1)(b)

- Framework Code: EU-AI-ACT
- Requirement Title: Technical Documentation for Providers of General-Purpose AI Models Referred to in Article 53(1)(a)

## Mapping Notes

A description of the model, including its general architecture and main parameters such as the number of parameters, context window size, input and output modalities.

SAFE-K8S ML-BOM captures model architecture metadata and parameter counts. Detailed model architecture documentation is an AI/ML domain deliverable, but SAFE-K8S enforces structured metadata in model registry entries.
