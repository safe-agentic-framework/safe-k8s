# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PW.4.1

- Framework Code: NIST-SSDF
- Requirement Title: Acquire and maintain well-secured software components from commercial, open-source, and other third-party developers.

## Mapping Notes

SAFE-K8S enforces vetting of third-party components at admission: image vulnerability scanning, signature verification, public model quarantine workflows, and approved source allowlists.
