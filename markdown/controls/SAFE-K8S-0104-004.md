# SAFE-K8S-0104-004 - Feature gate governance and lifecycle management

- Domain: D01 - Control Plane and Cluster Hardening
- Knowledge Area: 1.4 - CIS Benchmarks and Patch Management
- Maturity: Practitioner
- Classification: baseline

## Statement

Production Kubernetes clusters must maintain an inventory of all non-default feature gates, prohibit alpha-stage feature gates in production unless explicitly risk-assessed and approved, perform security impact assessment for beta-stage feature gates before enablement, and track feature gate lifecycle transitions across Kubernetes upgrades to disable deprecated or graduated gates.

## Objective

Prevent unintended security behavior from unstable or deprecated feature gates by governing which gates are enabled in production and tracking their lifecycle across Kubernetes version upgrades.

## Rationale

Feature gates control the activation of new, experimental, or deprecated Kubernetes behaviors. Alpha features are incomplete, potentially insecure, and may change without notice. Beta features may have undiscovered security implications. Graduated features that transition to GA may change default behavior, and lingering explicit gate settings can mask configuration drift. Without governance, feature gates accumulate over time, creating an opaque set of non-standard behaviors that complicate security assessment.

## Scope

Applies to all production clusters. Covers feature gates set on API server, controller-manager, scheduler, kubelet, and kube-proxy. For managed Kubernetes services, applies to customer-configurable feature gates and to tracking provider-managed default gate changes across version upgrades.

## Enforcement

- Primary Enforcement Point: Cluster Provisioning
- Implementation Surface: Cluster provisioning templates (feature gate flag configuration), change management process for feature gate modifications, and periodic feature gate inventory review during upgrade planning.
- Owner Primary: Platform Engineering
- Owner Supporting: Security Engineering; Architecture Review
