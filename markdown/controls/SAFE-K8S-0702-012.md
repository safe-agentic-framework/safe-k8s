# SAFE-K8S-0702-012 - Tenant namespace RBAC and admission boundary enforcement

- Domain: D07 - Storage, Multi-tenancy, and Resource Governance
- Knowledge Area: 7.2 - Namespace Isolation and Multi-Tenancy
- Maturity: Practitioner
- Classification: baseline

## Statement

Organizations must enforce tenant boundaries at the namespace and API/resource level by restricting tenant users and service accounts to namespace-scoped RBAC bindings and by using admission controls to block cross-tenant resource references and other noncompliant workloads before scheduling.

## Objective

Prevent cross-tenant API access and resource-manipulation paths by combining namespace-scoped RBAC with admission enforcement at workload creation time.

## Rationale

RBAC restricts who can act through the Kubernetes API, while admission control prevents workloads and resource definitions that would violate the tenant isolation model from ever being created. Either layer alone leaves exploitable paths for cross-tenant access, service impersonation, or policy bypass.

## Scope

Applies to all multi-tenant Kubernetes clusters. Each tenant namespace must have namespace-scoped RBAC bindings for tenant identities and admission policies that reject cross-tenant resource references and other isolation-model violations. Shared-service exceptions must be explicitly documented and reviewed.

## Enforcement

- Primary Enforcement Point: RoleBinding and ClusterRoleBinding review for cross-namespace grant detection, plus admission control policies enforced at create and update time.
- Implementation Surface: Namespace-scoped RoleBindings, restricted ClusterRoleBindings, service account binding reviews, admission control policies (Kyverno, OPA/Gatekeeper) preventing cross-namespace resource references, Pod Security Admission enforce mode configuration, image allowlisting, required label validation, and admission test cases that confirm rejection of violating resources.
- Owner Primary: Platform Engineering
- Owner Supporting: Security Engineering; Network Engineering
